# Running Cap'n Web on multiple runtimes with Hono

## Server

Cloudflare Workers:

```console
npm run dev:workers
```

Deno:

```console
npm run dev:deno
```

Bun:

```console
npm run dev:bun
```

Node.js:

```console
npm run dev:node
```

## Client

```console
npm run client
```

Batch:

```console
npm run client:batch
```
